# Dr. Link AI Powered Doctor Discovery Platform | By Team Rocket

**Dr. Link** is an intelligent, AI-powered doctor discovery platform that helps users find the right medical specialist by simply describing their symptoms in natural language. Built using **HTML, CSS, JavaScript, Flask** the platform uses along with **agentic workflows powered by n8n** to interpret needs, map them to the correct specialty, and provide **ranked doctor recommendations** with quick actions like **Call**.

The system also includes a lightweight **Doctor CRM Dashboard** with charts and insights, making it easy for admins to track doctor performance and user queries. Designed to be **fast, lightweight, and hackathon-optimized**, Dr. Link combines smart automation with a simple user-friendly interface to deliver an efficient healthcare search experience.
